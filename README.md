# boAt Website Clone

A static frontend clone of the **boAt Lifestyle** website built using **HTML** and **CSS**.
This project was created as part of my frontend learning journey while practicing dropdown menus, layout structuring, and ecommerce-style webpage design.

**Live Demo:**
[https://madhushreerasal.github.io/Boat-Website-Clone/](https://madhushreerasal.github.io/Boat-Website-Clone/)

# Features 

* Fixed navigation bar
* Dropdown and mega menu design
* Product showcase sections
* Promotional banners
* Ecommerce-style cards and layouts
* Multiple content sections
* Footer with categorized links
* Hover effects and interactive UI elements

# Technologies Used

* HTML5
* CSS3
* Remix Icon CDN

# What I Learned

While building this project, I practiced:

* Creating dropdown menus using CSS hover effects
* Using floats for webpage layouts
* Positioning elements with relative and absolute positioning
* Structuring large multi-section webpages
* Building reusable CSS classes
* Designing ecommerce-style interfaces

# Project Status 

This is a beginner-level frontend practice project and is currently **static and non-responsive**.
The main focus was improving HTML/CSS fundamentals and understanding webpage structure before learning modern layout systems like Flexbox and Grid.

# Future Improvements

* Make the website responsive
* Replace float-based layouts with Flexbox/Grid
* Improve accessibility and semantic HTML
* Add animations and transitions
* Optimize repeated code sections
